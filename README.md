In this folder contains the My Personal Event Planner app and this app allows you to add events so users can keep track of what eveents they have for the week or month. It contains a dashboard on the front where you can see all your events and you can delete any of them or load any of them or add another event. 
This Folder conatins several xml files with the activity_main file holding the nav bar for navigation of the app using the JetPack Navigation. The Fragment Add_Event where you will see the interface to add an event. The Last Interface is the Main Screen which is the Main Dashboard where you can see all your eveents and add or delete or load events.
The Folder has also some hava files where the AddEventFragment File Contains all the code for the AddEvent Fragment to work. The Event file conatining code to set out the event infroamtion of when the user puts information in it. The EventDao Containing some functions like delete and list and update. The Event List Adapter where it is used to be able to display the events on the main screen. Event OnClick is where the click functions are stored. Main Actviity is all the code for the jetpack navigation nav bar. The Main Screen is all the code for the screen to display events and delete events and load events and more. The Last File is RoomDB that is where the database is stored.
